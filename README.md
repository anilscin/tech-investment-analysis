# Tech Investment Analysis

A comprehensive investment analysis project focusing on Apple Inc. (AAPL) and Microsoft Corporation (MSFT).

## Overview
This repository contains detailed financial analysis, historical price data, and institutional holdings research compiled for academic purposes.

## Features
- In-depth financial analysis
- Historical price data visualization
- Institutional holdings research

## Tech Stack
- Python
- Pandas
- Matplotlib

## Usage
To run the analysis, ensure you have the required libraries installed. Use the following command:

```
pip install -r requirements.txt
```
